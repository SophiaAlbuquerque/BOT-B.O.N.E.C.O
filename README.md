# 🤖 B.O.N.E.C.O Bot

Bot de Discord focado em caos, diversão e progressão com sistema de XP, ranks e comandos interativos.

---

## 🚀 Funcionalidades

### 🎮 Comandos
- `!jogo` → Sugere um jogo aleatório e concede XP
- `!nivel` → Mostra seu XP e rank atual
- `!hoje` / `!hoje_tem` → Convoca geral com mensagem caótica

---

### 🧠 Sistema de Progressão

- Ganha XP ao usar comandos
- Sistema de ranks automático:
  - Camponês
  - Aventureiro
  - Veterano
  - Lenda

---

## 🛠️ Tecnologias

- Python 3
- discord.py
- python-dotenv
